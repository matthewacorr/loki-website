Assignment 5

      To determine what could be slowing down my webpage I used a website speed test, which emulates a user and takes note of anything that could be hindering your sites performance. Since this site is relatively small I haven't come across many issues related to slow webpage loading. I made the webpage snappy by limiting myself on the number of external libraries I used. The only external libraries I used were Font Awesome and Google Fonts which both promise that they keep your webpage fast. I also cut down on file size by compressing all the images in Assignment 1 and 2 as they take up a lot of valuable bandwidth. In the future I intend to split my CSS file into different sections so that I only need to load up the CSS I need and further increase the speed of my site.

      Before I redesigned my website for mobile I first shrunk down my browser window size to the size of a phone to determine what formatting was broken and what classes I had to change to get my design to look good on mobile. I opened a notepad and wrote down everything that needed fixing.

      For the media queries to work you need to set a width in which the content changes to mobile formatting, in order for me to find this I opened up Firefox's web development tools which allow you to dynamically change the size of the viewport; By doing this I was able to tell that my Social Media bar text started to overlap around 660px width. I used media queries to change font sizing to vw instead of px as it allows the text to resize with the browser. I also changed the amount of padding and margins depending on which device is being used as there is less room for whitespace on mobile and content is key.

      Another thing I had to consider was accessibility, I used a contrast checker to make sure that my background and foreground colors met the minimum for people with vision impairment. Another step I took was reducing the amount of clicking you need to do to navigate, my home page is a set of quick links to my assignments as those are the focus of the site. I also used bright colors for bold text and links as it helps them pop out from regular text. On hover links change from green to a bright orange which should hint the user that they are clickable.
