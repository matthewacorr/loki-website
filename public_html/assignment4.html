Assignment 4

      During high school I spent a couple of years doing very basic HTML and CSS design which taught me some of the fundamentals of making a functioning website. When designing this website, I had the goal of making it simple yet functional. While it might not be perfect, I have made all the HTML and CSS on this site except the CSS Reset (while checking w3schools a lot of course). I do not have screenshots of my website before I changed the CSS as I started changing it after assignment one; however it was very bland with only inline CSS style changes.

      First I went to coolors.co, a site which makes it easy to make a simple color palette, I played around with the colors and once I was happy with my selection I added the colors into my CSS. One of the issues I have currently is that I only have one CSS file and it has gotten progressively messier. In the coming days I aim to reduce the amount of redundant code and split it into multiple files for debugging as I keep running into inheritance issues.

      All the icons on this site are made with Font Awesome which is an easy way to add icons into your website without needing your own image file. Fonts were added through Google Fonts which provide fonts with no usage restrictions. The navbar and social media were made using inline unordered lists.
